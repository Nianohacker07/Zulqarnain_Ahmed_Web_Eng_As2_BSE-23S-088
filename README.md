## Project Description
This project is a client-side registration form developed using HTML, CSS, and JavaScript. It focuses on form validation and DOM manipulation without using external libraries.

## Features
- Clean and responsive registration form  
- Real-time form validation  
- Password strength and confirmation check  
- Gender selection using radio buttons  
- Country selection using dropdown menu  
- Terms and Conditions validation  
- Submit button enabled only when all fields are valid  
- Registration summary displayed after submission  

## Technologies Used
- HTML5  
- CSS3  
- JavaScript (DOM Manipulation)

## How to Run
1. Open the project folder  
2. Run the `.html` file in any web browser  
3. Fill in the form and submit after successful validation
